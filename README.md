## 🧠 Problem Statement

Modern Software Defined Vehicles (SDV) require:

- Multi-protocol backend services  
- Compliance with MISRA & ASPICE  
- Real-time dashboards  
- Faster development lifecycle  

Traditional development is slow, manual, and compliance-heavy.

---

## 💡 Proposed Solution

A **GenAI + RAG powered SoA Code Generator** that:

- Converts requirements → Production-ready service code  
- Generates HMI dashboards automatically  
- Ensures MISRA / ASPICE compliance  
- Integrates simulation validation (CARLA)  
- Accelerates SDV development by **40%**

## 🚀 Initial Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Harish-11-V/TATA-ELXSI-HACKATHON.git
cd TATA-ELXSI-HACKATHON
```

---

2️⃣ Create Virtual Environment (for AI Engine)

```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
```

---

3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

4️⃣ Start Docker Services

```bash
docker-compose up --build
```

---

5️⃣ Run GenAI Engine

```bash
python genai-engine/main.py
```
